This is a Rust codebase for my [Mix'n'Match tool](https://mix-n-match.toolforge.org/) for Wikidata. This includes the background task server, the webserver, and the HTML/JS interface.
Rationale is described in [my blog](http://magnusmanske.de/wordpress/?p=679).
